# COPY — тексты интерфейса

## 1) /start
Текст:
```
Привет. Круг друзей — это 4 ИИ-друга для сложных переписок, конфликтов и решений.

Скинь ситуацию, одно пересланное сообщение, войс до 10 минут или скрин — ребята сразу разберут.
Для пачки сообщений нажми «📎 Собрать переписку», кидай фрагменты подряд и потом жми «Готово».

Что получишь:
🧠 Ян — разложит факты и даст шаги.
❤️ Наташа — поможет понять чувства без осуждения.
🌀 Аня — поможет понять, что для тебя важно.
🎯 Макс — вернёт к реальности и скажет прямо.

Первый раз можешь просто написать ситуацию — разберут все четверо.
```
Inline-кнопки:
- `Не знаю с чего начать` (callback: `cs_help_start`)

Reply keyboard: основная клавиатура.

Первый текст после `/start` запускает PANEL сразу, free-first-panel сохраняется.

## 2) /help и /friends
Короткое описание:
- 🧠 Ян — разложит по полочкам и даст план
- ❤️ Наташа — поддержит и назовёт чувства
- 🌀 Аня — поможет понять, что для тебя важно
- 🎯 Макс — пошутит, вернёт на землю и отделит факты от эмоций

Инструменты:
- 📝 Напиши за меня — опиши ситуацию, и мы сформулируем сообщение кому угодно.
- 💬 Помоги ответить — перешли сложное сообщение, подскажем что ответить.
- 📎 Собрать переписку — включит режим для нескольких сообщений, войсов или скринов.
- 📋 Сводка — соберёт итог длинного разговора по команде или из вторичных действий.

## 3) Постоянная reply-клавиатура
- `🚀 Спросить всех` `📎 Собрать переписку`
- `📝 Напиши за меня` `💬 Помоги ответить`
- `👥 Друзья` `❓ Помощь`
- `⭐ Премиум` `⚙️ Настройки`

## 4) Выбор друга
- Первичный выбор: `Сейчас с тобой {Friend}.`
- Смена: `Сейчас с тобой {Friend}.`
- Если был pending panel: `Ок, отменил режим 🤝. Сейчас с тобой {Friend}.`

## 5) 🚀 Спросить всех (PANEL)
- Старт: `🤝 Ок. Кидай одну ситуацию, переписку, войс или скрин — разберут все четверо. Для пачки нажми «📎 Собрать переписку».`
- Повторный триггер в pending: `Я уже жду сообщение для всех друзей.`
- Перед генерацией: `Принял. Собираю разбор от всех друзей, это может занять до 20-30 секунд.`

## 6) 📎 Собрать переписку
- Старт: `📎 Кидай сообщения, войсы или скрины по очереди. Когда всё — нажми «Готово».`
- После фрагмента: `Принял 1 фрагмент. Кидай ещё переписку, войс или скрин. Когда всё — нажми «Готово».`
- `Готово`: запускает PANEL по собранному контексту.

## 6b) 📋 Сводка (SUMMARY)
- One-click запуск: `📋 Собираю сводку текущей сессии...`
- Формат: `📋 Сводка` + `Итого` + пункты + `Шаги`.
- Legacy alias: триггер `сводка` работает как синоним `итоги`.

## 7) 📝 Напиши за меня
- Вход без выбранного друга: `📝 Опиши, кому и что нужно написать: адресат, ситуация, желаемый тон. Можно одним сообщением, войсом или скрином. Разберут все друзья. Для пачки нажми «📎 Собрать переписку».`
- Вход с выбранным другом: `📝 Опиши, кому и что нужно написать: адресат, ситуация, желаемый тон. Можно одним сообщением, войсом или скрином. Ответит выбранный друг. Для пачки нажми «📎 Собрать переписку».`
- Перед SINGLE-генерацией: `📝 Принял. Собираю варианты формулировки...`
- Перед PANEL-генерацией: `📝 Принял. Собираю варианты формулировки от всех друзей, это может занять до 20-30 секунд.`

## 8) 💬 Помоги ответить
- Вход без выбранного друга: `💬 Вставь сообщение, на которое нужно ответить, и желаемый результат. Можно одним сообщением, войсом или скрином. Разберут все друзья. Для пачки нажми «📎 Собрать переписку».`
- Вход с выбранным другом: `💬 Вставь сообщение, на которое нужно ответить, и желаемый результат. Можно одним сообщением, войсом или скрином. Ответит выбранный друг. Для пачки нажми «📎 Собрать переписку».`
- Перед SINGLE-генерацией: `💬 Принял. Собираю варианты ответа...`
- Перед PANEL-генерацией: `💬 Принял. Собираю варианты ответа от всех друзей, это может занять до 20-30 секунд.`

## 10) Settings / privacy
- `⚙️ Настройки\nВыбери действие:`
- `/privacy`: что хранится и как удалить.
- `/reset`: подтверждение сброса текущей сессии.
- `/forget`: подтверждение удаления долгой памяти.
- `/demo`: пример PANEL-ответа.

## 11) Системные ошибки и сервисные тексты
- Rate limit: `Слишком быстро. Подожди пару секунд и напиши ещё раз.`
- Stale callback: `Эта кнопка устарела. Выбери ещё раз.`
- Generator failure: `Не удалось получить ответ от GPT. Попробуй отправить сообщение ещё раз.`
- Неизвестный callback/выбор: `Не понял выбор. Попробуй ещё раз.`

## 12) Share (после полезного ответа)
После PANEL/COMPOSE/REPLY/SUMMARY бот добавляет:
- Текст: `Что дальше?`
- Кнопка (callback `sh`): `Посоветовать бота`
- Ответ по нажатию: `Вот твоя ссылка для приглашения:\nhttps://t.me/<BOT_USERNAME>?start=ref_<code>`

## 14) Монетизация (UX-тексты в ToV бренда)

### Paywall (hard block, баланс = 0, после grace)
```
Друзья на паузе ☕

Бесплатные разговоры закончились. Пополни баланс, чтобы продолжить — ребята ждут.
```
Кнопки: `[📦 50 — 299₽]` `[📦 150 — 599₽]` `[📦 350 — 999₽]`

### Grace message (последнее сообщение при balance → 0)
После LLM-ответа, отдельным сообщением:
```
💬 Это было последнее сообщение. Чтобы продолжить разбираться вместе — пополни баланс:
```
Кнопки: `[📦 50 — 299₽]` `[📦 150 — 599₽]` `[📦 350 — 999₽]`

### Soft nudge (footer при balance ≤ 3)
Добавляется как footer к LLM-ответу:
```
💬 Осталось {N} — пополни, чтобы ребята были на связи.
```

### Balance footer (обычный, при balance ≤ 10)
```
💬 Баланс: {N} сообщений
```

### /balance — обычный пользователь
```
💬 Баланс: {N} сообщений
📊 Использовано: {used} сообщений

Пополнить:
```
Кнопки: `[📦 50 — 299₽]` `[📦 150 — 599₽]` `[📦 350 — 999₽]`

### /balance — bypass-пользователь
```
💬 У тебя безлимитный доступ ♾️
```

### /balance / ⭐ Премиум — монетизация выключена
```
💬 В текущей версии всеми функциями приложения можно пользоваться бесплатно. Это может измениться позже.
```

### Уведомление о зачислении (после успешной покупки)
```
✅ Баланс пополнен! +{amount} сообщений
💬 Баланс: {newBalance} сообщений
```
